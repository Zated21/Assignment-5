Important FAQ

What is Document Object Model(DOM)

DOM represents the structure of a wbpage in a tree format, we can use it to add,remove,modify and rearrange elements in a webpage

How do you select an element from DOM

we can select elements using DOM is multiple ways like getElementsByClass, getElementByID, getElementsByTagName and different types of querySelector

What is event delegation in the context of DOM, and why is it useful?

Event delegation is a pattern in which a single event listener is attached to a parent element, and that listener is used to handle events that occur on child elements

How do u manipulate an element's attributes and styles using the DOM?

In Dom, one can manipulate attributes and styles using the setAttribute, GetAttribute, removeAttribute, classlist.add/remove etc.
